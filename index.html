Forward-looking - agent action snapshot

Hugo is taking action on absenteeism and underperformance agent by agent. Distribution of statuses across all agents who had scheduled hours this month, broken out by site:

Status definitions. Active = absenteeism <6%. Probation = absenteeism 6-19.9%, daily monitoring, team-lead-driven. PIP Review = absenteeism ≥20% but agent is hitting both adherence and CSAT targets, performance improvement plan focused on attendance. Exit Candidate = absenteeism ≥20%, performance acceptable but not target-grade, attendance probation with consequences. Manage Out = absenteeism ≥20% and below adherence/CSAT target, performance review and exit plan over 1-2 months.

Agents requiring action

Agents with elevated absenteeism: Manage Out, Exit Candidate, and Probation cohorts.

Action snapshot
Manage Out - Performance review + exit plan over the next 1-2 months. Absenteeism ≥20% and below adherence/CSAT/AHT target. Top of the list goes through the Hugo PIP process; SA cohort coordinates with Mark / Jezreel for documentation given CCMA exposure.
Exit Candidate - Absenteeism ≥20% but performance is OK. Conversation first: attendance probation with clear consequences. Most of these are recoverable with management focus; the rest convert to Manage Out within 30 days if attendance does not improve.
PIP Review - Absenteeism ≥20% but adherence and CSAT both at or above target. These agents are showing up well when they are scheduled and producing good outcomes; the issue is attendance only. Performance improvement plan focused on attendance, not exit. We do not lose strong producers over schedule patterns we can fix.
Probation - Absenteeism 6-19.9%. Daily monitoring; team-lead-driven. Site abs % drops materially if even half of these move into Active.
